***************************
List of Epidemiology Models
***************************

This section documents the epidemiological model created by
Universities of Warwick and Lancaster. This model is used to produce a number
of research reports for the World Health Organisation (WHO) Strategic Advisory
Group of Experts on Immunisation (SAGE) Covid-19 Working Group.

It uses an extended version of an SEIR model and contact and region specific
matrices.
